###= Malicious Chrome Extensions & Data Theft

Recently, several browser extensions â€” including ad blockers and productivity tools â€” were discovered stealing sensitive user data. This shows that even trusted browser add-ons can become a security risk.

### What Happened?

Security researchers found that some Chrome extensions:

* Collected browsing history
* Read cookies and session tokens
* Accessed login pages
* Tracked user activity across websites

In some cases, the extensions were **sold to new developers** who then pushed malicious updates.
Users trusted the extension because it had millions of downloads and good reviews.

---

### Why This Is Dangerous

Browser extensions often request powerful permissions such as:

* "Read and change all your data on all websites"

This effectively gives them control similar to malware.
Attackers can use this to:

* Hijack accounts
* Track users
* Inject ads
* Redirect banking sessions
* Steal authentication cookies

This is especially dangerous because antivirus software usually does NOT detect malicious extensions.
